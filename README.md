# Prueba Rubén A. Martínez

Para ver el proyecto acceder mediante CMD a la ruta de una carpeta donde deseemos descargar el proyecto:
- cd <direción de la carpeta>
- git init
- git clone https://github.com/Rubenpard/avorisPrueba.git
- accedemos al proyecto y por CMD : npm install

## Caracteristicas

Es un proyecto en Angular version 17.3.10.

Se compone de un componente Padre y varios comopnentes hijos dentro de la carpeta "components":

HOME

- header-nav
- banner
- filters
- results
- footer

La pagina cuenta con una adaptacion responsive a Escritorio (1440px), Tablet (768px) y Movil (480px).

Cuenta con html semantico, metodologia BEM y procesado con sass.

Se han aplicado reglas de WCAG como navegacion por teclado en el header, añadido roles de navegación y title para accesibilidad.

Javascript basico como booleanos para despliegue de componentes y apliacación de directivas de Angular.
